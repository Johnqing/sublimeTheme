##sublime text2 theme使用##
1. 下载Theme - Flatland
2. copy以上文件夹，sublime中打开preferences -> Browse Packages放入其中
3. sublime中打开preferences -> User
4. 在其中修改下文配置
##User配置##
<pre>{
	"color_scheme": "Packages/Theme - Flatland/Flatland Dark.tmTheme",
	"font_size": 11,
	"ignored_packages":
	[
		"Vintage"
	],
	"theme": "Flatland Dark.sublime-theme"
}
</pre>
